Site De Commerce Electronique Nioyatech Account Management

Projet : Test du Site Web E-commerce

Description du Projet : Test des fonctionnalités et de l'expérience utilisateur du site Web de commerce électronique.

Méthodologies Utilisées : Modèle Scrum et méthodologie de test Agile.

Rédaction des Cas de Test : Des cas de test ont été rédigés pour des scénarios tels que l'inscription, la connexion, la recherche de produits, la sélection, l'ajout au panier, et le paiement.

Tests Manuel : J'ai effectué des tests manuels pour évaluer l'expérience utilisateur et les fonctionnalités.

Tests Automatisés : JJ'ai automatisé les tests en Java avec Selenium WebDriver et mis en place des tests fumée et parallèles avec TestNG (POM).

Suivi des Erreurs : J'ai utilisé JIRA pour signaler et suivre les problèmes identifiés.

Rapport : J'ai présenté les résultats des tests en utilisant l'outil de rapport Allure.

Processus Scrum : J'ai participé à la planification des sprints, aux réunions quotidiennes, aux revues de sprint et aux rétrospectives. Technologies Utilisées : Java, Selenium WebDriver, TestNG, JIRA, Allure.

Projet : Nioyatech Account Management

Description du Projet : Les processus de test de l'interface utilisateur (UI) et des API pour l'application NioyaTech Account Management.

Méthodologies Utilisées : Scrum model et méthodologie de test Agile.

Tests Manuel et Automatisés :

Tests UI : J'ai testé les scénarios utilisateur sur différents navigateurs en utilisant Selenium WebDriver.

Tests API : J'ai effectué des tests manuels avec Postman pour valider la fonctionnalité des services RestFul.

Scénarios de Test : J'ai automatisé les scénarios de test écrits en langage Gherkin.

Suivi des Erreurs : J'ai rapporté et suivi les problèmes identifiés en utilisant Jira Xray.

Rapports : J'ai présenté les résultats des tests en utilisant l'outil de rapport Cucumber.

Technologies Utilisées : Java, Selenium WebDriver, Postman, Jira Xray, Cucumber. CTLC (Software Testing Life Cycle) : Je participe à toutes les étapes des tests logiciels
