Project Name

E-commerce site UI testing

Description

Feb. 2024 - March 2024Feb. 2024 - March 2024

The project's objectives:

Test the functionality, usability and security of the e-commerce website. Identify website errors and weaknesses and provide suggestions for improvement.

Scope:

Functional tests:

User account creation and login process. Process of searching, selecting and adding products to cart. Payment operations and order finalization process.

Usability testing:

Evaluate whether the interface design is user-friendly. Ease and consistency of menu navigation. Compatibility of experiences across different browsers.

Testing tools:

Manual testing: Manual implementation of user scenarios. Automation Testing: Test scenarios developed in Java using Selenium WebDriver. Running smoke and parallel tests using the TestNG (POM) framework. Bug Tracking: Reporting and tracking bugs found using the JIRA bug tracking tool. Documentation: Created and maintained test cases, test plans, bug reports and other test documents.

Results:

Share the test results obtained at the end of the project and evaluate the suggestions for improvement presented.
